Document Type: schema
Databricks Catalog: prod
Databricks Schema: charge
Table Name: premodel


Description:
The table that is used in agreement discount application. This table merges the adjustments and corrections into the appropriate original charge rows.

Grain:
One row per (
    'lead_shipment_id',
    'shipment_id',
    'surcharge_id',
).

Generally this is one row per unique "charge".

Primary Use Cases:
- Asking questions about a client's spend.
- Asking questions about a particular shipment.

Notes:
- lead_shipment_number is the raw column value that the carriers provide, but lead_shipment_id is truly the id for the shipment. Note that the shipment identifier is lead shipment id.
tracking_number is the raw column value that the carriers provide, but shipment_id is truly the package id. 
so when a query asks for the shipment id, we give lead_shipment_id. the package is actually shipment_id.
- All charges from the same package will have the same shipment_id.
- All charges from the same shipment will have the same lead_shipment_id.
- Multipiece and hundredweight shipments have one lead_shipment_id, but one shipment_id for each unique package. 
hwt shipments bundle all the packages together and sum the weights for pricing. multipiece shipments are priced per package still.
queries about surcharge spend should filter out transportation charges.
- This table does not have surcharge id, so you will have to join on exact_matches to get the correct surcharge_id.

59. dim_divisor (DEC)

65. gross (DEC)

67. invoice_month (INT)

68. invoice_year (INT)

70. is_residential (BOOL)

71. minimum_charge (DEC)
  Description: the minimum charge of the shipment. No matter how good your regular discounts are on your transportation charge, your transportation charge cannot be lower than the minimum charge that the carriers specify for that particular service and zone. Note that you can also get discounts on your agreement to lower your minimum.

72. payor (STRING)

73. return_manager_id (STRING)

74. service (STRING)

75. tp_rate (STRING)

77. lead_shipment_id (STRING)
