Document ID: get_normalized_surcharge
Document Type: function
Input: Dataframe with table schema of premodel
Output: Datafarme with two extra columns: surcharge_id and surcharge_name
Function Name: get_normalized_surcharge

Description: 
For an input dataframe of charges, adds two columns: surcharge_id and surcharge_name to the dataframe. Use this function if the query is about surcharge information, but surcharge_id and surcharge_name is not in the shipments table.
Charge descriptions will be named differently in the carrier invoice files, but often they will be the same surcharge and priced the same. Getting the surcharge_id and surcharge_name for each charge is critical for identifying what the charge type actually is.


Example: 
Question: What would the normalized surcharge name be for the demand surcharge on tracking number xyz be?
Answer: (
    get_table("prod.charge.premodel")
    .where(F.col("tracking_number") == "xyz" and F.lower(F.col("charge_description")).like("%demand%"))
    .transform(get_normalized_surcharge)
    .select("surcharge_name")
    .limit(1)
)

Question: Give me the surcharge ids and surcharge names for the tracking number abc for staging.
Answer: (
    get_table("staging.charge.premodel)
    .where(F.col("tracking_number") == "abc")
    .transform(get_normalized_surcharge)
    ,select("surcharge_id", "surcharge_name", "charge_description", "*")    
)
