Function: get_table

Description:
Returns the appropriate table as a dataframe.

Arguments:
- table_name (string)
  The name of the table. the name will be catalog.schema.table_name.

Returns:
Snowpark DataFrame

Typical Use Cases:
- To get the table as a dataframe so the RAG can start applying other pyspark transformations or specific functions on it.

Example:
premodel = get_table("prod.charge.premodel")
