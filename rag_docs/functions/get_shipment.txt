Document ID: get_shipment
Document Type: function
Input: List[str] (tracking_number(s), lead_shipment_number(s), shipment_id(s), lead_shipment_id(s))
Output: Dataframe
Function Name: get_shipment

Description:
Returns shipment charge rows for a given tracking number.

Arguments:
- tracking_number (string)
  The carrier-provided shipment identifier.

Returns:
Snowpark DataFrame with the following columns:
- lead_shipment_id
- tracking_number
- shipment_date
- account_id
- service_type
- zone
- weight
- container_type
- surcharge_id
- charge_amount

Source Table:
prod.charge.premodel

Typical Use Cases:
- Find a shipment by tracking number
- Determine shipment attributes needed for repricing
- Join to agreements or rate tables

Example:
shipment = get_shipment("123456789")
